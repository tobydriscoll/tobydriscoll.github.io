Chebfun

Jun 6, 2019
Code Video Follow

Chebfun is a free MATLAB package and open source software project. Its aim is to provide numerical computation with functions. My interest in and contribution to the system is most visibly in the solution of ordinary and (1+1 dimensional) partial differential equations. Using familiar MATLAB syntax such as \ and eigs, solutions to boundary-value or eigenvalue problems can be obtained with full numerical precision automatically.
