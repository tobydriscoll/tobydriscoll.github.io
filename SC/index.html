The Schwarz-Christoffel Toolbox for MATLAB

The SC Toolbox is a problem-solving environment for computation and interaction with conformal maps to regions bounded by polygons, including unbounded regions, logical quadrilaterals, and channels. It includes a module for the solution of the Laplace equation on such regions with piecewise constant boundary conditions, to ten digits or more in seconds. Nearly all of the Toolbox functions are accessible through a graphical interface.

The software is distributed in zip archives. Please refer to the user's guide .

For MATLAB 7 and later: The current version Unpack to a directory in your MATLAB path. The user's guide is not included in this distribution. See also my book for usage of the toolbox and much more about SC maps in general.

For MATLAB 6: The last version includes a function for quickly solving Laplace's equation on polygonally bounded regions. The user's guide is included in this distribution. Unpack to a directory in your MATLAB path.

For MATLAB 5: Download version 2.1 of the toolbox and the corresponding user's guide.

For MATLAB 4: Version 1.4 of the toolbox is available as a zip file. It runs best under version 4.2 of MATLAB. You may also use version 4.1 of MATLAB (version 4.0 under MS Windows), but the graphical user interface will be crippled.  There is a user's guide for this version.
