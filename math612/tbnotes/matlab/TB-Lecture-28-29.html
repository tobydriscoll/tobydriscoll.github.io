Lectures 28 and 29: The Francis QR algorithm¶

Simultaneous iteration¶

If we perform power iteration simultaneously on independent vectors, and $A$ has eigenvalues of distinct magnitudes, then the content of each vector is dominated by the eigenvectors sorted in order. QR factorization thus reveals these eigenvectors in order, and we can approximately diagonalize $A$ as the iteration continues.

A = magic(6)

A =

     3    32     7    21    23    25

     8    28    33    17    10    15
    30     5    34    12    14    16
     4    36    29    13    18    11

We symmetrize the matrix and find a similar tridiagonal substitute.

In [2]:
A = hess(A+A');
lambda = eig(A);
[~,idx] = sort(abs(lambda));
lambda = lambda(idx)

    5.8213
    8.8418

We'll track progress by means of the sizes of the subdiagonal elements.

In [3]:
Q = eye(6);
subdiag = [];
for k = 1:30
    Z = A*Q;
    [Q,R] = qr(Z);  % Q converges to eigenvectors
    D = Q'*A*Q;     % D converges to diagonal
    subdiag = [subdiag; diag(D,-1)'];
end
semilogy(abs(subdiag),'-o')
axis tight
title('Subdiagonal terms in SI')
xlabel('k'), ylabel('|a^{(k)}_{j+1,j}|')

The graph confirms that we are converging to all the eigenvalues linearly.

Unshifted QR iteration¶

The unshifted QR iteration is algebraically equivalent to SI.

In [4]:
D = A;
subdiag = [];
for k = 1:30
    [Q,R] = qr(D);
    D = R*Q;
    subdiag = [subdiag; diag(D,-1)'];
end
semilogy(abs(subdiag),'-o')
axis tight
title('Subdiagonal terms in QR')
xlabel('k'), ylabel('|a^{(k)}_{j+1,j}|')

Shifted QR iteration¶

As stated above, the SI/QR iteration finds eigenvalues in order sorted by magnitude.

In [5]:

D =

  222.0000   -0.0000    0.0000    0.0000    0.0000    0.0000
   -0.0000   74.8884  -10.2952   -0.0000    0.0000    0.0000
         0  -10.2952  -68.5239    0.0000   -0.0000    0.0000
         0         0    0.0000  -21.0276   -0.0000    0.0000
         0         0         0   -0.0000    8.8418   -0.0000
         0         0         0         0   -0.0000    5.8213

This fact is exploited by using the lower right element in a shift-and-invert strategy.

In [6]:
D = A;  m = length(A);
subdiag = [];
for k = 1:8
    mu = D(m,m);
    [Q,R] = qr(D - mu*eye(m));
    D = R*Q + mu*eye(m);
    subdiag = [subdiag; diag(D,-1)'];
end
semilogy(abs(subdiag),'-o')
axis tight
title('Subdiagonal terms in shifted QR')
xlabel('k'), ylabel('|a^{(k)}_{j+1,j}|')

We're seeing cubic convergence for one eigenvalue. (The values are impossibly small because we haven't maintained true symmetry, thereby dodging the effects of roundoff.)

Deflation¶

Once an eigenvalue is found in the lower right corner, the eigenvalue problem for the upper left submatrix can be decoupled, leading to the recursive technique of deflation. Note that the subdiagonal is already substantially reduced, so the remaining eigenvalues tend to be found quickly.

In [8]:
D = D(1:m-1,1:m-1);  m = m-1;
mu = D(m,m);
[Q,R] = qr(D - mu*eye(m));
D = R*Q + mu*eye(m)
