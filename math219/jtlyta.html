Introduction¶

Data was taken from two data sets, one from BechdelTest.com and the other from TheNumbers.com. Data was collected from 1,794 movies released from 1970 to 2013 to observe the relationship between the presentation of women in a film and the film's profits. Gender bias in films is denoted by whether or not each film passes the Bechdel test. The Bechdel test states that if "there are at least two named women in the picture, they have a conversation with each other at some point, and that conversation isn’t about a male character," then the film is considered as passing.

import numpy as np

quant_columns = ["budget", "domgross", "intgross", "budget_2013$", "domgross_2013$", "intgross_2013$"]

def count_outliers(x):
    Q25, Q75 = x.quantile([.25, .75])
    I = Q75 - Q25
    outliers = (x < Q25 - (1.5 * I)) | (x > Q75 + (1.5 * I))
    outliers_sum = outliers.sum()
    return outliers_sum

outliers_count = bechdel[quant_columns].apply(count_outliers)

outliers_count = outliers_count.rename(index=dict(zip(quant_columns, quant_columns)))

There are three columns in the data that represent periods of time: year, period code, and decade code. period code represents five-year intervals starting with 1-2 (2010-2015) and ending at 8-9 (1970-1975). decade code represents ten-year intervals, or decades, starting at 1.0-2.0 (2010-2020) and ending at 5.0-6.0 (1970-1980).

The three line plots below show the percentage of movies passing the Bechdel Test when grouped by each of these columns. The first graph is more sensitive to outliers, which makes it difficult to see a trend. However, once we break down the data by periods and then decades, we can see a steady increase in percentages over the years. The decade with the most increase in test passes is 3.0-4.0, or the 1990s. After 2010, there is a slight decrease, but over 40% of movies are still passing.
